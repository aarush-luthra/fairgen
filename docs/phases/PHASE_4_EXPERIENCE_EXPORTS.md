# Phase 4: Experience and Exports

## Goal

Complete the core user workflows across data preview, charts, reports, downloads, config persistence, and AI-assisted regeneration.

## Scope

- Data table pagination
- Recharts visualizations
- Fairness report cards
- Download CSV
- Download fairness report JSON
- Save/load/share config
- API connection test
- Demo scenario loader

## Done When

- The three user journeys are runnable through the UI.
- Before/after chart differences are easy to interpret.
- Users can save config and export results without touching code.
