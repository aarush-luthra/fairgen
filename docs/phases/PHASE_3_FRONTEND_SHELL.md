# Phase 3: Frontend Shell

## Goal

Build the main dark-theme interface shell with sticky score header, configuration panel, and tab structure.

## Scope

- React app scaffold
- Sticky header
- Config panel with collapsible bias sections
- Tooltip system
- AI prompt box
- Tabbed main area
- Loading overlay and staged status messages

## Done When

- A user can adjust config, trigger generation, and navigate the main tabs.
- The live fairness score remains visible in the header.
- The interface is usable on mobile and desktop.
