# FairGen Project Brief

## Vision

Build an approachable but technically credible application that helps teams generate synthetic credit datasets with stronger demographic fairness characteristics for model prototyping, demos, and education.

## Core Promise

"Generate synthetic credit data. Remove bias. Build fairer AI."

## Primary User Types

1. Data scientists who want bias-reduced synthetic training data.
2. Business analysts who want to explore fairness intuitively.
3. Hackathon judges who need to understand the product quickly.

## Product Pillars

1. Zero-explanation defaults.
2. Clear before/after comparisons.
3. A visible fairness score that moves with user actions.
4. Plain-English explanations for every technical concept.
5. A conversational AI constraint layer.

## Delivery Strategy

1. Build backend correctness first.
2. Add generation pipeline and AI integration.
3. Layer in polished frontend workflows.
4. Finish with exports, demo mode, and presentation polish.
